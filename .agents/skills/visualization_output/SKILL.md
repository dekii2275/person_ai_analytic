# Skill: Visualization and Output

## Vai trò

Vẽ kết quả và ghi output video.

## Draw only

```text
person bounding box
confidence score
```

Không thêm UI phức tạp.

## Required artifact

```text
outputs/output.mp4
```

## Verification

Sau khi ghi xong phải mở lại video bằng code và xác nhận:

- opens successfully;
- width đúng;
- height đúng;
- FPS hợp lệ;
- có ít nhất một frame đọc được.

Không được chỉ kiểm tra file tồn tại.
